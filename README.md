## 🚀 Featured Projects

### 🎓 [ManhajLM](https://github.com/almunzir99) — *AI-Powered Exam Platform for Arab Students*
> **Problem:** Sudanese national exam content exists only as scanned Arabic PDFs, inaccessible to AI-driven tutoring. **Solution:** Built an end-to-end pipeline — Azure Document Intelligence OCR → Gemini 2.0 Flash extraction → canonical JSON schema — transforming paper exams into structured, AI-queryable content powering an interactive tutor.

`#SemanticKernel` `#Gemini` `#pgvector` `#.NET10` `#Flutter` `#ArabicNLP`

---

### 🌍 [Mubasher](https://github.com/almunzir99) — *Offline-First NGO Field Operations Platform*
> **Problem:** NGO field teams in Africa operate in low/no-connectivity environments with no reliable tooling. **Solution:** Built a config-driven, offline-first mobile platform with a multilingual NLU engine (AR/EN/FR), OCR-assisted document pipeline, and a state-machine workflow engine — zero code changes needed to deploy new operational modules.

`#Flutter` `#OfflineFirst` `#NLU` `#OCR` `#StateMachine` `#ConfigDriven`

---

### ⚖️ [Al-Wafi AI](https://github.com/almunzir99) — *RAG Legal Intelligence System*
> **Problem:** Legal firms managing 500K+ files have no intelligent way to query document content. **Solution:** Designed and built an internal RAG application with an advanced OCR pipeline and document chunking strategies, enabling natural-language queries over a massive legal document corpus.

`#RAG` `#.NET` `#Flutter` `#DocumentIntelligence` `#LegalTech`
